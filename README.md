# XT-LastMemberOnline

Mit diesem AddOn kannst du dir ein Widget erstellen in dem die Benutzer angezeigt werden die in denx Stunden online waren. Die Anzahl der Stunden kannst du in den Widget Einstellungen selber festlegen.
